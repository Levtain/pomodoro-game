# 番茄钟养成游戏 - 完整设计方案

## 📖 项目概述

### 核心理念
**"自律的痛苦" → "奖励的快乐"**

通过**"钓鱼场景"包装**，将番茄钟时间管理转化为：
- 专注时间 = 钓鱼机会
- 完成专注 = 获得奖励（鱼/渔具）
- 长期坚持 = 收集图鉴、解锁新水域

---

## 🏛️ 三权分立架构

### 上议院（主会证 - 我）
- **角色：** 协调、战略、决策、最终审批
- **权力：** 最终审批权、整体方向把控
- **任务：** 
  - 协调各院讨论
  - 整合意见
  - 最终决策
  - 通知执行层

### 长老院（3个Agent + 3个频道）
- **角色：** 专业质疑、找茬、提供建议

#### 长老A：战略大师
- **人格：** 战略眼光、市场洞察、行业敏感
- **核心使命：** "这个方向没救了，市场已有巨头"
- **说话风格：** 专业、沉稳
- **质疑重点：** 市场规模、竞争格局、护城河、长期价值
- **口头禅：** "从战略角度..."
- **工作方式：** 找行业专家观点、对比巨头、质疑战略可行性

#### 长老B：竞品大师
- **人格：** 产品大师、用户体验敏感、竞品对比专家
- **核心使命：** "你看过Forest的设计吗？人家做得比你好十倍"
- **说话风格：** 专业、尖锐、数据支撑
- **质疑重点：** 产品定位、用户体验、竞品功能、用户留存
- **口头禅：** "从产品角度..."
- **工作方式：** 分析竞品、找差异化、强调用户价值

#### 长老C：技术大拿
- **人格：** 技术大拿、工程化思维、直白不客气
- **核心使命：** "你们这些帮人根本不懂技术"
- **说话风格：** 直白、粗暴、技术至上
- **质疑重点：** 技术选型、架构设计、工程化程度、技术债务
- **口头禅：** "React？你脑子进水了？应该用Electron"
- **工作方式：** 从底层工程角度质疑、找技术漏洞、强调可维护性

### 下议院/执行层（子会证）
- **角色：** 执行具体任务
- **任务：** 执行上议院分配的任务

## 🎣 核心设计：钓鱼场景包装

### 心理机制转变

#### 传统模式（惩罚式）
```
自律 = 痛苦
不做 = 罪恶感
坚持 = 强迫自己
时间 = 压力
```

#### 钓鱼场景包装（奖励式）
```
自律 = 钓鱼
专注 = 钓到鱼
完成 = 获得鱼
坚持 = 解锁新水域
时间 = 快乐（因为知道有回报）
```

## 🎮 游戏系统设计

### 1. 三层任务系统

#### 日常任务（轻量级）
**目的：** 保持用户活跃度、日常习惯

**任务类型：**
- 每天完成1个番茄钟（25分钟）
- 每天查看3次应用
- 每天签到（20金币）

**奖励：**
- 每个番茄钟 = 10金币
- 连续7天签到 = 额外50金币 + "勤奋称号"
- 完成所有日常 = 100金币

**要求：**
- 无（自动触发）
- 每天重置

#### 长线任务（中量级）
**目的：** 培养长期使用习惯、目标感

**任务类型：**
- 累计完成100个番茄钟 = 解锁"深水域"
- 累计完成500个番茄钟 = 解锁"远海"
- 连续30天登录 = "老渔夫"皮肤

**奖励：**
- 每个里程碑 = 大量金币（100-500）
- 特殊称号
- 专属皮肤

**要求：**
- 累计进度永久保存
- 一次性奖励

#### 收集任务（重量级）
**目的：** 增强收集欲和成就感

**任务类型：**
- 收集30种不同鱼（每种普通5条）
- 收集10套装饰皮肤
- 收集5个稀有渔具

**奖励：**
- 收集完30种鱼 = "图鉴大师"称号
- 收集完10套皮肤 = 专属"彩虹皮肤"
- 收集完5个渔具 = "收藏家"称号

**要求：**
- 无进度要求
- 一次性奖励

### 3. 钓鱼机制

#### 基础规则
- 每次1个番茄钟完成 = 获得1次钓鱼机会
- 钓鱼机会在番茄钟开始后触发（专注25分钟）
- 每次钓鱼随机钓到一种鱼
- 鱼的种类和价值完全随机

#### 概率控制
- 普通鱼：30%概率
- 稀有鱼：15%概率
- 史诗鱼：5%概率
- 鲸鱼：5%概率（如果幸运）

#### 保底机制
- 连续10次没钓到稀有鱼 = 必得1次稀有鱼
- 连续30次没钓到史诗鱼 = 必得1次史诗鱼
- 每周至少1次保底稀有鱼

#### 钓鱼动画
- 开始：抛竿动画
- 过程：浮漂提示
- 钓到：拉线动画+震动反馈
- 没钓到：空钩动画
- 钓到稀有：放大特效+音效
- 史诗鱼：全屏特效

- 稀有鱼：10/10

- 集完普通鱼：解锁图鉴详情

- 收集半数鱼：解锁特殊背景

## 📝 总结

### 核心创新
1. **"钓鱼场景"包装** - 弱化自律痛苦，强化奖励快乐
2. **三层任务系统** - 日常（活跃）+ 长线（成长）+ 收集（成就）
3. **多稀有度鱼类** - 30%普通 + 15%稀有 + 5%史诗 + 5%传说
4. **保底机制** - 避免"非酋"挫败感
5. **图鉴系统** - 长期目标和收集欲

### 商业模式
- **免费+内购** - 核心功能免费，装饰和加速付费
- **月度订阅（可选）** - 9.9元/月，加速50%
- **一次性付费** - 传说皮肤、高级渔具

### 竞争优势
- **差异化明显** - 不是单纯的番茄钟
- **粘性强** - 图鉴收集、长线目标
- **心理健康** - 奖励而非惩罚
- **商业潜力** - 皮肤、加速、订阅
